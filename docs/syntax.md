## Syntax

Some parts of Rend, such as operators and keywords, have specific syntax that must be followed. Below is a list of the various operators, keywords, and types used in Rend.

### Operators

* **;**
* **{}**
* **()**
* **=**

### Binary operators

* **+**
* **-**
* **\***
* **/**
* **%**
* **<<**
* **>>**

### Logic operators

* **&&**
* **||**
* **!**

### Comparison operators

* **==**
* **>**
* **>=**
* **<**
* **<=**
* **!=**

### Keywords

* **return**
* **if**
* **else**

\* if can be chained after else like **else if**

#### Types

* **int** = an unsigned 32-bit integer.
* **bool** = a boolean value stored as a single byte (1 or 0).
